# Expense Tracker (MERN Stack)

A full-stack Expense Tracker application built using the MERN stack to help users manage their income and expenses through an intuitive dashboard.

## Features

- Secure user authentication (JWT)
- Add, edit, and delete transactions
- Income and expense tracking
- Category-wise expense management
- Budget tracking
- Monthly financial summaries
- Interactive charts and analytics
- Search and filter transactions
- Responsive user interface

## Tech Stack

**Frontend**
- React.js
- Tailwind CSS
- Axios

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB

**Authentication**
- JWT
- bcrypt

## Installation

```bash
git clone https://github.com/ManognaKodukulla/expense-tracker-mern.git
```

```bash
cd client
npm install

cd ../server
npm install
```

Run the frontend and backend servers to start the application.
